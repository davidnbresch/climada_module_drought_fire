README: NASA FIRMS MODIS Thermal Anomalies/ Fire Locations Download
This zip file supplied contains the data for the requested dates in your area-of-interest. Depending on the date range selected you may receive 1 or 2 files containing:

i) _NRT files (e.g. firmsxxxxxNRT.shp, firmsxxxxxNRT.csv) contain MCD14DL data: : Near Real-Time (NRT) MODIS Thermal Anomalies / Fire locations processed by NASA FIRMS, part of the Land Atmosphere Near real-time Capability for EOS (LANCE).

and/or

ii) _MCD14ML files (e.g. firmsxxxx_MCD14ML.shp, firmsxxxx_MCD14ML.csv) contain MCD14ML Standard/science Quality MODIS Thermal Anomalies / Fire locations processed by the University of Maryland with a 3-month lag and distributed by NASA FIRMS. These data replace the MCD14DL when available.

Both MCD14ML and MCD14DL are processed using the same MOD14 Fire and Thermal Anomalies Algorithm but the NRT version (MCD14DL) uses predicted (attitude and ephemeris) geolocation and the standard / science quality version (MCD14ML) uses definitive (attitude and ephemeris) geolocation.

The difference between MODIS geolocation when using predicted vs. definitive attitude and ephemeris is routinely less than 100 meters. However, there are situations, particularly before and after spacecraft maneuvers and during space weather events, when the difference can increase up to several kilometers. The recommendation from EOSDIS is to reprocess data when the definitive attitude and ephemeris data become available.

The reason the files are split is to ensure users clearly distinguish between these two data sources. Should you wish to combine the datasets you will still be able to distinguish the source using the Collection. Version 5.0 indicates the data is the NRT version of Collection 5 data. Version 5.1 is the standard/science quality version of Collection 5. (See FIRMS FAQ, "What are Collections?" https://earthdata.nasa.gov/data/near-real-time-data/faq/firms For more information regarding the attribute fields, please visit the following page: https://earthdata.nasa.gov/data/near-real-time-data/faq#firms20 Visit the NASA FIRMS project website at http://earthdata.nasa.gov/firms

PROJECTION INFORMATION
The MODIS Thermal Anomalies/ Fire locations supplied to you are in the WGS-84 geographic projection (the "latitude/longitude projection", EPSG:4326). For help on importing the data into your local GIS, please refer to our FAQ at https://earthdata.nasa.gov/data/near-real-time-data/faq.

IMPORTANT
For further information, please refer to the latest version of the MODIS Fire Users Guide, which can be found at: https://earthdata.nasa.gov/data/near-real-time-data/firms/publications and the FAQ section (https://earthdata.nasa.gov/data/near-real-time-data/faq).

Please note that there are data missing from several of the data sets. There is data missing from end of June to the beginning of July in 2001, 2002 is missing some data throughout the data set, 2007 has some missing data from mid August and data is missing for part of 21 April 2009, and missing for 22 April 2009. There might also be some erroneous data present in the data set. Please refer to the disclaimer below.
